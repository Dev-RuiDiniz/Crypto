# 10 - API

## Endpoints principais
- Health: `/api/health`, `/api/health/db`, `/api/health/worker`
- Runtime: `/api/balances`, `/api/orders`, `/api/mids`, `/api/events`
- Config: `/api/config`, `/api/bot-config`, `/api/bot-global-config`, `/api/config-status`
- Credenciais: `/api/tenants/<tenantId>/exchange-credentials*`

Veja também `docs/openapi.yaml` (quando aplicável ao escopo coberto).
